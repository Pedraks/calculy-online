Rescisão: Salário R$ 2.000,00

Cálculo específico para salário base de R$ 2.000,00.

Entenda a rescisão de quem ganha R$ 2.000,00

Se o seu salário base registrado em carteira é de R$ 2.000,00, o cálculo das suas verbas rescisórias deve partir desse valor exato. Muitas calculadoras genéricas pedem para você digitar, mas aqui já configuramos a base correta para evitar erros.

O que está incluído no cálculo para R$ 2.000,00?

  • Aviso Prévio: O valor de um mês de aviso será exatamente R$ 2.000,00 (se for indenizado), mais 3 dias por ano trabalhado.
  • 13º Salário: Cada mês trabalhado no ano dá direito a 1/12 de R$ 2.000,00.
  • Férias: O valor base das suas férias será R$ 2.000,00 acrescido de um terço.

Dúvidas Comuns para essa faixa salarial

O INSS muda para quem ganha R$ 2.000,00?

Sim. O desconto do INSS segue a Tabela Progressiva 2025. O simulador acima já aplica a alíquota correta automaticamente sobre o valor de R$ 2.000,00.

Tenho direito a Seguro Desemprego ganhando R$ 2.000,00?

Provavelmente sim, se foi demitido sem justa causa. O valor das parcelas será calculado com base na média dos últimos salários (próximo a R$ 2.000,00). Use nossa Calculadora de Seguro Desemprego para confirmar.

*Nota: Este cálculo considera R$ 2.000,00 como salário bruto base. Adicionais como horas extras e comissões devem ser somados à média para o cálculo final exato.
